# piletop

A live CPU core heatmap. Just pure color.

## Installation

```
uv tool install git+https://github.com/jooooow/piletop.git 
```

## Usage

```
piletop
```

![alt text](image.png?v=2)

## Keys
- `q`: Quit
- `t`: Switch theme

## Options
- `-i`: Set refresh interval (default is 0.5 seconds)
